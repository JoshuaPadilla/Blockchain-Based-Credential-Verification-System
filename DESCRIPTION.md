# Project Description

Blockchain-Based Credential Verification System is a full-stack platform for issuing, signing, and validating academic credentials with tamper-evident blockchain proofs. It uses a NestJS + PostgreSQL backend for application data, a React frontend for institution and verifier workflows, and a Solidity smart contract (`CredentialVerifier`) to anchor credential hashes, enforce signer authorization, and support revocation/restoration checks.

## One-Line Description

A full-stack academic credential platform that combines PostgreSQL-backed records with on-chain verification and multi-signer authorization.

## Short Description (GitHub About)

Issue, sign, and verify academic credentials using NestJS, React, PostgreSQL, and Solidity smart contracts for tamper-evident on-chain integrity.
